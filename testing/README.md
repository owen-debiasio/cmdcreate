# Testing the Features of cmdcreate

## Files

- ./install.sh
  - Builds cmdcreate and installs it to `/usr/local/bin/`
- ./uninstall.sh
  - Does the opposite of `install.sh`, removes it from `/usr/local/bin/`
- ./testing/main.py
  - Tests features for cmdcreate **(See below)**

---

## Running tests

### Available tests

- Creating commands
- Removing commands (forced and not forced)
- Editing commands
- Listing commands/favoriting commands
- Searching for commands
- Displaying command contents
- Renaming commands
- Repairing commands
- Importing/Exporting commands
